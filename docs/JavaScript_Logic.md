# JavaScript Logic Implementation

## Objective

The JavaScript layer provides all client-side functionality for the FinWise AI project.

---

## Features Implemented

- Form Validation
- EMI Calculation
- Credit Score Analysis
- DOM Manipulation
- Event Handling
- Dynamic Result Rendering

---

## Credit Score Module

JavaScript validates the entered credit score and categorizes it into:

- Excellent
- Good
- Poor

It also displays personalized recommendations dynamically.

---

## EMI Calculator Module

JavaScript accepts:

- Loan Amount
- Interest Rate
- Loan Tenure

It calculates:

- Monthly EMI
- Total Interest
- Total Payable Amount

using the standard EMI formula.

---

## DOM Manipulation

JavaScript updates page content dynamically without refreshing the browser.

---

## Validation

The application validates:

- Empty fields
- Invalid values
- Range checking

before calculations.

---

## Event Handling

JavaScript uses event listeners for:

- Button Click
- Form Submission
- Reset
- Input Validation

---

## Conclusion

JavaScript provides the complete client-side business logic for the application and improves user experience through dynamic calculations and real-time validation.
