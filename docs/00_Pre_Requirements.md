## Required Technologies

The following technologies will be used throughout the development of the FinWise AI project:

### Flask
Flask is used as the backend framework to develop REST APIs and manage server-side application logic.

### HTML, CSS and JavaScript
These technologies are used to design and develop the responsive user interface of the application.

### Claude AI
Claude AI will be integrated to analyze financial information, evaluate loan eligibility, and generate intelligent financial recommendations.

### Git
Git is used for version control, allowing project changes to be tracked and managed efficiently.

### Node.js and npm
Node.js and npm are used to manage frontend packages and support the web development workflow.
